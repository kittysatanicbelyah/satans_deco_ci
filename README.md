Wanted curtains? You're on the right address, soul!

![cool curtains, cooler curtains](https://raw.githubusercontent.com/kittysatanicbelyah/random_bu-shi_go/refs/heads/main/deco/ci/cibanner2.png)

This mod adds curtains and pelmets:
- 16 dyed curtains;
- 16 dyed curtains with edge trim;
- 16 dyed beaded curtains;
- 16 dyed blinds.
- 16 dyed solid pelmets
- 16 dyed soft pelmets
- 32 dyed ornate pelmets
- 11 wooden solid pelmets
- 11 carved solid pelmets

Curtains are based on iron bars block, so it can bend to your will.
Pelmets, on other hand, are based on vines, so you can place them on any side of a block.

What can you do?

- Build something cool;
- Open and close curtains with redstone power or right hand;
- Place pelmets on blocks, curtains, bars, fences, walls.

What config does?

- When curtains are triggered for opening or closing code checks all the connected curtains. In order to prevent theoretical crash exploit there is a limit for amount of checked curtains.

# About Satan's deco

This is a multi-mod project to create a lot of somewhat unique and interesting decorations.

For simplicity all mods of this project are called in same fashion:
```java
"Satan's deco : *enter_name*"
```
