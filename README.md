# Satan's deco: Curtainsanity

## About Satan's deco

This is a multi-mod project to create a lot of somewhat unique and interesting decorations.

For simplicity all mods of this project are called in same fashion: 
```java
"Satan's deco : *enter_name*"
```
It currently supports only 1.20.1, but 1.21.1 and 26.x are planned

## About Curtainsanity

This part of Satan's deco adds curtains. 
- 16 dyed curtains.

# Where Can I download mod?
You can check out releases on [github](https://github.com/kittysatanicbelyah/satans_deco_ci/releases)

Modrinth and Curseforge is currently unsupported, but there are plans for them in near future.
